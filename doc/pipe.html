Private method pipe

Pipes a reconciliation from parent to child

Parameters:

  • parent must be an Object.
    (Promise)

  • child must be an Object.
    (Promise)

Returns an Undefined
(undefined)

function pipe ( parent, child ) { parent.then( function ( arg ) { child.resolve( arg ); }, function ( e ) { child.reject( e ); } ); }
