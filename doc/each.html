Private method each

Iterates obj and executes fn Parameters for fn are 'value', 'index'

Parameters:

  • obj must be an Array.
    (Array to iterate)

  • fn must be a Function.
    (Function to execute on index values)

Returns an Array
(Array)

function each ( obj, fn ) { var nth = obj.length, i = -1; while ( ++i < nth ) { if ( fn.call( obj, obj[i], i ) === false ) { break; } } return obj; }
